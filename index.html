CONTENIDO:

  1. Estructura WEB HTML5

    2. Además de definir partes individuales de una pagina (como un párrafo o una imagen), HTML5 también dispone de elementos de bloque que se pueden utilizar para estructurar las distintas áreas de un sitio web. Al utilizar estas etiquetas para tipos de contenido específico, se mejora la legibilidad y facilidad de reutilización tanto del código HTML5 como de las hojas de estilo. Una página web típica lleva la siguiente estructura semántica:

    Algunas de las etiquetas distintivas de una página HTML5 estructurada:


  2. article (Artículo)

    3. Encuadra un bloque de contenido independiente, que tiene sentido por sí mismo aparte del resto de la página, puede contener otros elementos article y section.


  3. section (Sección)

    4. Es parecido o similar al elemento article, pero se usa para agrupar cada parte de la página que constituye una sección en sí misma, más grande, o contenido principal, puede contener elementos article y section. Se sugiere iniciar éstas con un título de encabezado (h1, h2...).


  4. aside (Lateral)

    5. Incluye contenido que no está directamente relacionado con el contenido principal (secundario), pero que puede aportar información adicional relacionada indirectamente con él (resúmenes, biografías del autor, enlaces relacionados, etc.), suele ponerse a un lado del documento.


  5. footer (Pie)

    6. Representa un grupo de contenido al final de la página: (Direcciones, Autor, licencias, fechas, etc.)
